# Earthquakes
На карте отображены землетрясения и их магнитуда.
<br>
Данные за 24.11.2021- 24.01.2022 по штату Калифорния ,и всему миру. 
<br>
Источник данных csv и json файлы с earthquake.usgs.gov. Визуализация в plotly.
